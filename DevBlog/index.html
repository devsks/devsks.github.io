How to use '%n' in C?

You may be knowing to trick to calculate the length of a string using printf() method but below I have discussed a far better approach.

In order to find the numbers of characters in a string, we can use flag characters in C.

The [%n] format identifier stores the numbers of characters read by the input stream (i.e stdin) to the given address. Eg:

#include <stdio.h>
  ﻿int main()
  ﻿{
      char st[123];
      int len;
      
      scanf("%s%n", st, &len); //length is getting stored at location of len
      printf("The length of the entered string is %i.", len);
      
      return 0;

  }

Input: devsks

Output:

The length of the entered string is 6.

Manual page states that: Nothing is expected from '%n'; instead, the number of characters consumed thus far from the input is stored through the next pointer, which must be a pointer to int. This is not a conversion, although it can be suppressed with the * flag.
After the scanf() statement has completed the length of the string is saved in the address of len variable(i.e &len).

We can also use this identifier to count the numbers of digits of a number and speed up our program runtime etc.
